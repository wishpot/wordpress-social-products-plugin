== Description ==

Tested with **Wordpress 3.1**

This free plugin allows you to display ads from various product categories on your wordpress blog. They can be displayed on pages or in widgets.
The ads are generated by the plugin. The generated widgets differ in size (300x600, 300x250, 160x600, 120x600 etc.).


If you have problems or questions, please post it in the **[Support forum](http://wordpress.org/tags/wishpot?forum_id=10#postform)** .
